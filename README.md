# React web app with openstreetmap that displays POIs for electric cars for hire.

### Technologies:
* fix filter

## Technologies:
The project was created with:
* Typescript
* React
* styled-components
* React Leaflet
* OpenStreetMap (ODbL)

## Scripts:
Install dependencies:
```
 npm install
```
Start dev mode:
```
 npm start
```
Run tests:
```
 npm test
```
Build App:
```
 npm build
```
